# Miniverse Template

Template for developing and exporting a Miniverse game.

## Quick Start

1. Create a new repository based off this template
2. Rename the project by executing the [rename.py](rename.py) script

## Pull Template Changes

To pull the latest changes from the template repository, execute the following command:

```bash
git remote add template https://github.com/qkabi-games/miniverse-template.git
git fetch --all
git merge template/master --allow-unrelated-histories
```
